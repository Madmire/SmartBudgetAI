# *BUDGET AI*  

Une application web innovante pour la gestion des budgets destinée aux étudiants.  

## *Description*  
*BUDGET AI* permet aux étudiants de suivre leurs dépenses et leurs revenus de manière simple et efficace tout en recevant des conseils personnalisés grâce à un chatbot alimenté par l'IA. Cette application vise à améliorer la gestion financière des étudiants et les aide à maintenir un équilibre budgétaire sain.  

---

## *Fonctionnalités principales*  
- *Suivi des dépenses et revenus :*  
  Enregistrez vos transactions quotidiennes et visualisez vos flux financiers.  
- *Conseils générés par IA :*  
  Utilisez notre chatbot pour recevoir des recommandations personnalisées sur la manière de gérer votre budget.  
- *Statistiques et graphiques :*  
  Analysez vos finances grâce à des graphiques interactifs générés par *Chart.js*.

## *Installation*  

Pour exécuter l'application en local, suivez les étapes suivantes :  

1. *Clonez le projet :*  
   bash
   git clone https://github.com/username/budget-ai.git
   cd budget-ai
     

2. *Installez XAMPP :*  
   Téléchargez et installez [XAMPP](https://www.apachefriends.org/index.html).  

3. *Configurez la base de données MySQL :*  
   - Lancez XAMPP et démarrez Apache et MySQL.  
   - Importez le fichier SQL (budget_ai.sql) dans *phpMyAdmin* pour créer la base de données.  

4. *Lancez l'application :*  
   - Déplacez les fichiers dans le dossier htdocs de XAMPP.  
   - Accédez à l'application via http://localhost/projetianisql/pages/.
